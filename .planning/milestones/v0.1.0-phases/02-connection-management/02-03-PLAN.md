Required ConnectionManager API (consumed by src/index.ts and Phase 3 tool handlers):
```typescript
export class ConnectionManager {
  constructor(config: AppConfig);

  // Start all account connections concurrently. Errors in individual accounts
  // are caught and logged — connectAll() itself never throws.
  connectAll(): Promise<void>;

  // Returns the live ImapFlow client for the account, or a structured error.
  // Phase 3 tool handlers call this before issuing any IMAP command.
  getClient(accountId: string): ImapFlow | { error: string };

  // Returns the connection status for a named account.
  // Used by Phase 3 list_accounts tool.
  getStatus(accountId: string): AccountConnectionStatus | { error: string };

  // Gracefully closes all connections. Uses Promise.allSettled — one failure
  // does not prevent others from closing.
  closeAll(): Promise<void>;
}
```

Shutdown pattern (from research):
```typescript
// Use Promise.allSettled so one failed logout does not block others
async function shutdown(manager: ConnectionManager): Promise<void> {
  logger.info('Shutting down — closing IMAP connections');
  await manager.closeAll();
  process.exit(0);
}
process.on('SIGTERM', () => { void shutdown(manager); });
process.on('SIGINT',  () => { void shutdown(manager); });
```
