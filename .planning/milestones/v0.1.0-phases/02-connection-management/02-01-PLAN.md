    - Test: 'error' event on ImapFlow is handled (does not throw uncaught exception)
    - Test: shutting-down flag prevents reconnect loop from starting during gracefulClose

    connection-manager.test.ts:
    - Test: connectAll() starts a connection attempt for each account in config
    - Test: getClient() returns ImapFlow instance when account is 'connected'
    - Test: getClient() returns structured error { error: string } when account is 'reconnecting'
    - Test: getClient() returns structured error { error: string } when account is 'failed'
    - Test: getClient() returns structured error { error: string } when account_id is unknown
    - Test: one account failing to connect does not prevent other accounts from connecting
    - Test: closeAll() calls gracefulClose() on all connections using Promise.allSettled
