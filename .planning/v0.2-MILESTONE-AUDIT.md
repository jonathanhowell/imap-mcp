### Behavioral Issues (Non-blocking)

**1. `from` field format inconsistency (Phase 9)**
- `read_messages` + `read_message` assemble `from` as bare address instead of `"Name <addr>"` format from `formatAddress` (introduced Phase 7). Agents see inconsistent formats for the same sender across tools.
- Fix: Apply `formatAddress` in `src/tools/read-message.ts` and `src/tools/read-messages.ts`.

**2. `keywords` not populated in `list_messages` (Phase 11)**
- `listMessages()` fetches `flags: true` but does not extract keywords. `searchFolder` (search-service.ts:113) does. `get_new_mail` / poller path is unaffected.
- Fix: Add `keywords: [...(msg.flags ?? new Set<string>())].filter((f: string) => !f.startsWith("\\"))` to the `listMessages` return mapping.
