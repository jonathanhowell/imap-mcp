## Decisions Made

- `messageFlagsRemove([uid], [keyword], { uid: true })` mirrors the `messageFlagsAdd` pattern from flag_message — consistent IMAP API usage
- `removeKeyword` uses `.filter()` rather than mutation to produce a clean new array — semantically correct for removal
- Idempotent by nature: filtering an absent keyword returns the same array unchanged
