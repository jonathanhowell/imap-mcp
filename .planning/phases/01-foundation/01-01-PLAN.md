<task type="auto" tdd="false">
  <name>Task 1: Initialize package.json and install dependencies</name>
  <files>package.json</files>
  <action>
    Run `npm init -y` to create package.json, then set `"type": "module"` (required for ESM + Node16 module resolution). Install all dependencies in one pass:

    ```
    npm install @modelcontextprotocol/sdk zod yaml
    npm install -D typescript @types/node@22 vitest eslint typescript-eslint prettier husky lint-staged
    ```

    Then edit package.json to add scripts:
    ```json
    {
      "type": "module",
      "scripts": {
        "build": "tsc",
        "lint": "eslint src/ tests/",
        "lint:fix": "eslint src/ tests/ --fix",
        "format": "prettier --write \"src/**/*.ts\" \"tests/**/*.ts\"",
        "test": "vitest run",
        "test:watch": "vitest",
        "test:coverage": "vitest run --coverage",
        "prepare": "husky"
      }
    }
    ```

    Add lint-staged config to package.json:
    ```json
    {
      "lint-staged": {
        "*.ts": ["eslint --fix", "prettier --write"]
      }
    }
    ```

    The `prepare` script runs `husky` automatically on `npm install` to set up the git hooks directory.
  </action>
  <verify>
    `node -e "const p = JSON.parse(require('fs').readFileSync('package.json','utf8')); console.error(p.type === 'module' ? 'OK' : 'FAIL')"` prints OK. Also verify `node_modules/@modelcontextprotocol/sdk` exists.
  </verify>
  <done>package.json has type:module, all runtime and dev dependencies installed, scripts defined, lint-staged config present.</done>
</task>

<task type="auto" tdd="false">
  <name>Task 2: Configure TypeScript, ESLint, Prettier, Vitest, and Husky</name>
  <files>tsconfig.json, eslint.config.mjs, .prettierrc, vitest.config.ts, .husky/pre-commit, .gitignore</files>
  <action>
    Create `tsconfig.json` with strict mode and Node16 module resolution (required for `.js` import extensions in ESM):
    ```json
    {
      "compilerOptions": {
        "target": "ES2022",
        "module": "Node16",
        "moduleResolution": "Node16",
        "outDir": "./build",
        "rootDir": "./src",
        "strict": true,
        "esModuleInterop": true,
        "skipLibCheck": true,
        "forceConsistentCasingInFileNames": true,
        "declaration": true,
        "sourceMap": true
      },
      "include": ["src/**/*"],
      "exclude": ["node_modules", "build"]
    }
    ```

    Create `eslint.config.mjs` using ESLint 9 flat config (NOT .eslintrc — that format is removed in ESLint 9):
    ```javascript
    import tseslint from "typescript-eslint";

    export default tseslint.config(
      ...tseslint.configs.recommended,
      {
        rules: {
          "no-console": "error",
          "@typescript-eslint/no-explicit-any": "warn",
          "@typescript-eslint/explicit-function-return-type": "off",
        },
      },
      {
        ignores: ["build/**", "node_modules/**"],
      }
    );
    ```

    Create `.prettierrc`:
    ```json
    {
      "semi": true,
      "singleQuote": false,
      "trailingComma": "es5",
      "printWidth": 100,
      "tabWidth": 2
    }
    ```

    Create `vitest.config.ts`:
    ```typescript
    import { defineConfig } from "vitest/config";

    export default defineConfig({
      test: {
        environment: "node",
        include: ["tests/**/*.test.ts"],
        coverage: {
          provider: "v8",
          reporter: ["text", "json"],
        },
      },
    });
    ```

    Initialize Husky and create pre-commit hook:
    ```bash
    npx husky init
    ```

    Write `.husky/pre-commit`:
    ```sh
    #!/usr/bin/env sh
    npx lint-staged
    npx vitest run --reporter=verbose
    ```

    Create `.gitignore`:
    ```
    node_modules/
    build/
    *.js.map
    coverage/
    .env
    ```

    Create empty placeholder directories for future phases:
    ```bash
    mkdir -p src/config src/connections src/services src/tools tests
    ```
  </action>
  <verify>
    Run `npm run lint` — should exit 0 (no source files yet, but config must be valid). Run `npm test` — should exit 0 (no test files yet, Vitest exits cleanly with no tests). Run `npx tsc --noEmit --version` to confirm TypeScript is accessible.
  </verify>
  <done>All config files exist. `npm run lint` exits 0. `npm test` exits 0. `npm run build` would succeed once src/ has TypeScript files. Pre-commit hook installed in .husky/pre-commit.</done>
</task>
